Необходимо написать функцию, которая определяет монотонность слайсла. Если слайс растет или уменьшается монотонно, вывести true, иначе false.

Пример работы программы:

Input:
{1, 2, 3, 4}
Output:
true

Input:
{1, 1}
Output:
true

Input:
{9, 5, 1}
Output:
true

Input:
{9, 5, 10}
Output:
false
